# Acknowledgements

## W3C Specifications

The `xslt` and `xquery` CLI tools implement W3C and Qt4 Community Group specifications:

- [XSLT 3.0](https://www.w3.org/TR/xslt-30/) / [4.0](https://qt4cg.org/specifications/xslt-40/Overview.html)
- [XQuery 3.1](https://www.w3.org/TR/xquery-31/) / [4.0](https://qt4cg.org/specifications/xquery-40/xquery-40.html)
- [XPath 3.1](https://www.w3.org/TR/xpath-31/) / [4.0](https://qt4cg.org/specifications/xpath-functions-40/Overview.html)

We thank the W3C XSL Working Group, the W3C XML Query Working Group, and the
Qt4 Community Group for their work on these specifications.
